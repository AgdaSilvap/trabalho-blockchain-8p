Trabalho realizado para a disciplina de Laboratório Redes de Computadores - Blockchain


Dependências:
1. node v24.11.1
2. npm v11.6.2
3. extensão do metamask instalada no navegador

Como executar:
1. clone o projeto deste repositório localmente na sua máquina
2. crie um arquivo .env, copie o relacionado em .env.exemple e altere para os dados do seu projeto supabase
3. execute ```console npm install ``` para instalar as principais dependências do projeto
4. execute ```npm run dev``` para executar o projeto no navegador
